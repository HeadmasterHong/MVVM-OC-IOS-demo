# 轻量级跨平台框架

## 简介

lightart是一个轻量级的跨平台解决方案，其特点为：

1. 支持所有平台：Android/iOS/Web/WXApp/QuickApp；
2. 数据驱动的原生实现；
3. 支持脚本语言

## 目录结构


lightart
     
     |-- android    原生Android框架实现
     |-- ios        原生iOS框架实现
     |-- web        web框架实现
     |-- wxapp      微信小程序实现
     |-- quickapp   快应用实现
     |-- editor     lightart编辑器
     |-- docs       文档说明
     |-- tools      工具
